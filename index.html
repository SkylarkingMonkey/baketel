Tipple Book

www.tipplebook.com

Tipple Book is my first web-based application. It is a tool for bartenders and bar managers to quickly generate cocktail recipes with an easy to read format that can be saved to a list and easily shared with a unique URL. Plans to expand this include users being able to save their cocktails into a personal library. This app is built using HTML, CSS, Javascript, Python and SQL.
